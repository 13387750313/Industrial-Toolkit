📖 视觉系统选型指南

📏 尺寸测量
适用于精密零件尺寸检测，要求高精度、高稳定性。
推荐相机 500万以上 GigE
推荐镜头 FA镜头 / 远心镜头
推荐光源 同轴光 / 背光
精度要求 ≤ 0.01mm
🔍 表面缺陷检测
适用于金属、塑料等产品表面划伤、凹坑、污渍等检测。
推荐相机 200万以上 USB3.0
推荐镜头 FA镜头 + 同轴光
推荐光源 同轴光 / 条形光
检测方式 明场 / 暗场
📝 字符/二维码识别
适用于产品追溯、标签读取、激光码识别等场景。
推荐相机 200万以上 USB3.0
推荐镜头 FA镜头
推荐光源 环形光 / 同轴光
字符大小 ≥ 0.5mm
🎯 定位引导
配合机械手实现抓取、装配等自动化引导。
推荐相机 200万以上 GigE
推荐镜头 FA镜头
推荐光源 环形光 / 条形光
重复精度 ≤ 0.05mm

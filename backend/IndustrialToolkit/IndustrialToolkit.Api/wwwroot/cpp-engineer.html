Modbus协议说明
Modbus是一种串行通信协议，广泛应用于工业自动化领域。支持RTU和TCP两种传输模式。
OPC UA协议说明
OPC UA (Open Platform Communications Unified Architecture) 是一个工业通信协议，用于工业自动化和设备之间的数据交换。
Ethernet/IP协议说明
Ethernet/IP 是ODVA开发的工业以太网协议，用于在工业自动化设备之间进行实时通信。
PROFINET协议说明
PROFINET 是基于工业以太网的开放自动化标准，由PROFIBUS国际组织开发。
